A hypothesis testing framework for the ratio of means of two negative binomial distributions

Page last updated by Matthew Denwood on 16th October 2019

OVERVIEW

This website accompanies the article "A hypothesis testing framework for the ratio of means of two negative binomial distributions: classifying the efficacy of anthelmintic treatment against intestinal parasites" by Denwood et al. that is currently under peer review. A pre-peer-reviewed version of the article is available to download from the arXiv database and can be cited as:

M.J. Denwood, G.T. Innocent, J.C. Prentice, L. Matthews, S.W.J. Reid, C.B. Pipper, B. Levecke, R.M. Kaplan, A.C. Kotze, J. Keiser, M. Palmeirim, and I.J. McKendrick (2019). A hypothesis testing framework for the ratio of means of two negative binomial distributions: classifying the efficacy of anthelmintic treatment against intestinal parasites. arXiv:1910.06667 [stat.ME]. Retrieved on 2019-10-16, from the arXiv database: https://arxiv.org/abs/1910.06667

For the R and C++ code that underlies the methods presented, see the GitHub repository for the bayescount package for R. The development version of the R package can also be installed (without the need for C++ compilers) from our Drat repository using the following R code:

install.packages('bayescount', dependencies=TRUE, repos=c(CRAN="https://cran.rstudio.com/", kuawdc="https://ku-awdc.github.io/drat/"))

If running the R package locally then a good place to start from is the vignette:

vignette("bayescount", package="bayescount")

DATA ANALYSIS

An interactive application is provided for analysis of an existing dataset using the five methods presented in the article. The output is presented in terms of the 95% confidence intervals or p-values produced by each method, along with the relevant typology and resulting classification according to our framework. In the future this application will be extended to provide a validation of the type I and type II error rates associated with each of the analysis methods using parameter values estimated from your data. To launch the application from within R on your computer use:

bayescount::launch_shiny("data_analysis")

Alternatively, the same interactive application can be run from a remote server here.

TYPOLOGIES

This tool allows the user to explore the consequences of using different interpretative frameworks along with the five methods presented in the article in terms of the classification probabilities associated with a dataset simulated with given population parameters. To launch the application from within R on your computer use:

bayescount::launch_shiny("typologies")

Alternatively, the same interactive application can be run from a remote server here.

STUDY PLANNING

An interactive application is provided for planning a study based on parameter values provided by the user. The output is presented in terms of the relative frequency of each typology and resulting classification given a specified sample size. The primary purpose of this tool is to estimate the necessary sample size for a new study, but the type I and type II error rates are also provided for comparison of each of the five methods of analysis presented in the article. To launch the application from within R on your computer use:

bayescount::launch_shiny("study_planning")

Alternatively, the same interactive application can be run from a remote server here.
